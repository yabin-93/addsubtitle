# Test Data

Use this directory for fixed test assets and generated temporary files.

Recommended layout:

- `upload_files/video/`: source videos such as `英文30S.mp4`
- `upload_files/audio/`: source audio such as `audio.mp3`
- `upload_files/image/`: images such as `cover.png`
- `temp/`: runtime-generated temporary files

Example filenames expected by later upload tests:

- `test_data/upload_files/video/英文30S.mp4`
- `test_data/upload_files/audio/audio.mp3`
- `test_data/upload_files/image/cover.png`
